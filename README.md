
# UserReviews

Программная система для определения степени объективности пользовательского рейтинга на 
сайтах-агрегаторах (для примера используется "КиноПоиск")

## Описание системы

Все модули такой системы: получение данных, обработка данных, предсказание (НС) и
визуализация - работают с MongoDB и независимо друг от друга. Каждый модуль использует
свой конфигурационный файл со всей необходимой информацией.   
Однако, модуль визуализации также взаимодействует ещё и с Memcached для хранения кратковременной информации.

## Зависимости

Данная система подключается к MongoDB по пути, указанному в конфигурационных файлах.   
Кроме того, перед запуском работы программы требуется установить все необходимые библиотеки
из файла `requirements.txt`.
